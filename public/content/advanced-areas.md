# 심화 교육축

공식 홈페이지에서는 **2개 심화 교육축**이라는 표현을 기본으로 사용합니다. 사용자가 “트랙”이라고 질문하면 같은 의미로 안내하되, 대외 표시는 “심화 교육축”을 우선 사용합니다.

## 1. 건축디자인·디지털트윈 AI

건축 정보와 디지털모델을 기반으로 설계 대안을 탐색하고 설계 의사결정을 다루는 심화 교육축입니다.

### 주요 내용

- 공간 데이터와 건축정보 분석
- 디지털모델링과 설계 의사결정
- BIM/CIM, 디지털트윈, 생성형 설계 활용
- 디지털모델 기반 설계안 탐색
- 생성형 설계와 성능 기반 검토
- 공간 데이터 기반 설계 의사결정
- 프리팹·OSC 설계 개념

## 2. 구조안전·시공품질·건축환경 AI

현장 중심 데이터와 물성 정보를 활용해 AI로 구조·시공·안전·품질·건축환경 판단을 다루는 심화 교육축입니다.

### 주요 내용

- 구조·시공·안전 데이터 분석
- 품질·검측·건축환경 성능 정보 활용
- 센서·비전 기반 모니터링과 예측진단
- 구조 모니터링·예측 진단
- 시공·안전·품질 데이터 분석
- 건축환경 성능 분석과 검토
- 현장 보고·문서·센서 정보 활용
