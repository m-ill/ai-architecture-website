AI건축융합학과 홍보 플랫폼 & 검색엔진 고도화

학과 핵심 인재상 및 교육 철학
"건축 정보를 읽고, AI와 데이터로 판단·설계·개선하는 융합형 책임기술 인재 양성"

본 플랫폼은 단순 웹 구축에 그치지 않고, 검색엔진 지수가 부족한 수험생 Q&A를 정밀 스코어링하여 억측을 제어하고, 외부 인공지능(LLM)이 학과의 정책 원문을 정확하게 해석·활용할 수 있도록 설계된 차세대 **NLWeb-lite** 규격을 탑재하고 있습니다.

사람을 위한 웹 인터페이스 (Human Web)

수험생과 학부모가 직관적으로 학과의 비전을 느끼고, 정보에 쉽고 빠르게 접근할 수 있는 프리미엄 레이아웃을 구현했습니다.

  • 동적 퀵 칩 스택: 자주 묻는 질문 10가지를 suggested_questions.json을 통해 비동기 바인딩하여 탐색 유연성 확보.
  • 라이트박스 갤러리: 건축 인재상 그래픽 및 공식 안내 브로셔를 클릭하면 블러 글라스 오버레이로 중앙 확대하여 모던한 감성을 시각화.
  • 반응형 모션 설계: Outfit & Inter 폰트 적용 및 HSL 가변 컬러, 호버 효과를 통한 몰입감 넘치는 미학적 인터랙션 선사.
인공지능을 위한 웹 규격 (Machine-Readable NLWeb)

ChatGPT, Claude, Gemini, Perplexity 등 최신 AI 검색 크롤러와 챗봇이 왜곡(환각 현상) 없이 학과 데이터를 정식 훈련할 수 있도록 설계했습니다.

  • /llms.txt 규격 수립: AI 서비스 봇을 전담 마킹하는 로드맵 파일을 구현해 학과 정보의 출처 신뢰성을 극대화.
  • 통합 마크다운 제공 (canonical.md): 자연어 형태로 가공된 학과 정보 전체 원문을 원클릭 텍스트 단위로 학습할 수 있게 적재.
  • JSON 구조화 데이터: 입학, 교과과정, 졸업 후 진로 등 고품질 관계형 스키마 데이터를 public/data/ 경로에 정밀 분할 배치.
Q&A 검색 엔진 다중 가중치 정밀 스코어링 시스템

수험생이 입력한 엉성하거나 오타가 섞인 자연어 키워드를 구조화된 108개 FAQ 데이터베이스와 연계하는 자체 매칭 스코어링 모델입니다. Edge API (서버)와 Client Fallback (브라우저) 간의 점수 산출 공식이 100% 일치하여 대칭적 정확성을 보장합니다.

질문명(Question) 완전 일치 +10,000점
질문명(Question) 부분 포함 / 서브스트링 매칭 +8,000점
유사 질문(Aliases) 완전 일치 +5,000점
유사 질문(Aliases) 부분 포함 +3,000점
질문 토큰 일치 (개당 +1,000점) / 키워드 단어 포함 (개당 +1,000점) +1,000점
답변 본문(Answer) 검색 매칭 / 카테고리 일치 보정 +500 ~ +1,000점
검색 신뢰성 기준 제어 (Confidence Threshold): 누적 점수가 1,000점 미만인 경우 비관련 정보로 자동 필터링하여, answer_policy.json에 공식 명시된 Fallback 메시지를 송출해 환각 오류를 완벽 차단합니다.
입학처 억측 차단 (Grade Safety Limit)

신설 학과인 만큼 예측이 불가하거나 편향될 수 있는 수능 등급컷, 내신 평균선, 경쟁률 질문에 대해 AI 챗봇이 단정적으로 말하는 것을 정책적으로 금지합니다.

  • 안전 장치 키워드 감지: 내신, 수능, 등급, 합격선, 경쟁률 등이 인입될 시 시스템이 긴급 차단 코드를 발동합니다.
  • 안전 답변 강제 매핑: 해당 감지 시 입학처 공식 안전 답변 가이드라인인 adm-021, adm-022, adm-023 문항에 **+6,000점의 초고화질 가중치**를 순간 주입하여 억측 답변을 완벽 차단합니다.
동의어 반응성 보정 (Synonym Sync)

수험생이 학부모의 관습적인 표현인 트랙이라는 단어로 질문을 던지면, 플랫폼 공식 권장 용어인 **심화 교육축**의 모든 FAQ 문항에 자동으로 **+4,000점의 보정치**를 수여하여 유연하고 똑똑한 교과과정 매칭을 주도합니다.

입학 디스클레이머 자동화 및 옐로우/골드 UI

수집된 입학 관련 데이터를 보여줄 때는 2027학년도 대입 시행계획 기준임을 명시하고 최종 모집요강을 반드시 재확인하도록 알립니다.

시스템은 입학 관련 범주의 답변에 answer_policy.json의 경고 조항을 자동 접합하며, 본문에 '모집요강' 키워드가 중복 포함된 경우 이를 지능적으로 생략해 메시지의 피로도를 제어합니다.

1-Click 로컬 개발 가동 배시 스크립트

윈도우 개발자의 편의성을 최우선으로 고려하여, 콘솔 명령어를 입력할 필요 없이 더블 클릭 한 번으로 모든 로컬 개발 서버가 작동하는 환경을 배포했습니다.

  • 스크립트명: start_dev.bat (루트 디렉터리 위치)
  • 무설정 기동: 실행 시 node_modules의 유무를 스스로 감지하고 누락 시 npm install을 스마트하게 우선 자동 설치 수행.
start_dev.bat 코드 구조 
@echo off
title AI건축융합학과 NLWeb-lite 로컬 개발 서버
echo ===================================================
echo  [AI건축융합학과] 로컬 개발 서버를 기동합니다.
echo ===================================================
if not exist node_modules (
    echo [안내] 최초 의존성 패키지를 설치 중입니다...
    call npm install
)
call npm run dev
pause
CI/CD 자동 배포 및 도메인 관리

플랫폼이 깃허브 원격 리포지토리와 긴밀히 맞물려 동작합니다.

  • 저장소 연동 완료: https://github.com/m-ill/ai-architecture-website 에 안전하게 배포 준비 코드가 완벽 푸시되었습니다.
  • 실시간 갱신 시스템: 깃허브 저장소에 코드를 git push하는 즉시 Cloudflare Pages 빌드 파이프라인이 즉각적인 운영서버 교체 및 라이브 릴리스를 완수합니다.
