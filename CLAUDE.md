# Project Instructions

This project is a Korean admissions/promotional website for AI건축융합학과.

## Never change the canonical message

Use this framing consistently:

> 건축 정보를 읽고, AI와 데이터로 판단·설계·개선하는 융합형 책임기술 인재 양성

## Do not overclaim

- Do not say admission, employment, architect license, or professor participation is guaranteed.
- Admission data is based on the 2027 admissions implementation plan and must include a final 모집요강 confirmation notice.
- Company names are examples only.

## Preferred terminology

Use “심화 교육축” in public UI. Accept “트랙” as a user query synonym.

## Data source of truth

- `public/data/faq.json` for Q&A
- `public/data/department.json` for core identity
- `public/data/admissions.json` for admissions
- `public/content/canonical.md` for AI-readable long form
